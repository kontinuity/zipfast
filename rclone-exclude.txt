# rclone exclude-from for zipfast
#
# Sends only the source needed to build the Docker image. The image build itself
# rebuilds node_modules (pnpm install), the SPA (vite build -> web/dist), the
# Prisma client (prisma generate -> web/src/prisma), and the Go binaries.
#
# Example:
#   rclone sync ./ remote:zipfast            --exclude-from rclone-exclude.txt
#   rclone sync ./ sftpremote:/srv/zipfast   --exclude-from rclone-exclude.txt
# Then on the server:  docker build -t zipfast .   (or: docker compose up -d --build)
